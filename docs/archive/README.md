# Archive

This directory contains internal planning, migration, and intermediate architecture documents from the Octify → Crucue migration and early feature development phases.

These documents are preserved for historical reference but are **not part of the public-facing documentation**. The current, accurate architecture and product documentation lives in the `docs/` root. If a topic was removed from `docs/` at the repo root, its last snapshot is typically **here** under `docs/archive/` (for example `crucue_architecture.md` in this folder, superseded by [`final_production_architecture.md`](../final_production_architecture.md)).

| Archived file | What it was |
|---------------|-------------|
| `crucue_migration_plan.md` | Phased migration plan from Octify to Crucue |
| `crucue_feature_slice_02.md` | Internal feature slice planning doc |
| `crucue_product_direction.md` | Early product strategy notes (canonical narrative today: [`about-crucue.md`](../about-crucue.md)) |
| `crucue_architecture.md` | Intermediate architecture doc (superseded by `final_production_architecture.md`) |
| `crucue_ai_architecture.md` | Early AI architecture notes (superseded by `gemma4_strategy.md`) |
| `crucue_voice_architecture.md` | Voice pipeline planning (superseded by architecture doc) |
| `crucue_voice_data_model.md` | Voice data model notes (superseded by `firebase_data_model.md`) |
| `crucue_gemma4_voice_strategy.md` | Early Gemma 4 + voice strategy (superseded by `gemma4_strategy.md`) |
| `crucue_firestore_schema.md` | Early Firestore schema reference (superseded by `firebase_data_model.md`) |
